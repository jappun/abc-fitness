# ABC Ignite Feature Demo
## What it does
- inform members how busy the gym is
- predicts how busy the gym will be at a given time and day

Check out the demo video here: https://youtu.be/3_3UuiuFhew or run the project on your own device.

## Running Locally
Clone the repo via terminal
```
git clone https://github.com/jappun/abc-fitness
```
Move into the project folder
```
cd abc-fitness
```
Install dependencies
```
npm instsall
```
Run the project
```
npm run dev
```
You should now be able to open the project on http://localhost:5173/
